PRoot.PROPERTY_KEY_INPUT_SOURCES Field

The key that identifies a change in the set of this root's InputSources.

protected static readonly object PROPERTY_KEY_INPUT_SOURCES;

Remarks

In a property change event the new value will be a reference to the list of this root's input sources, but the old value will always be null.
