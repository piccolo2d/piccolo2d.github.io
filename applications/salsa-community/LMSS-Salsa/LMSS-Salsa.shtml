The Salsa Project

The Salsa project is creating a zoomable information workspace using Jazz, and includes the following major activities:

  • The DARPA Interactive Image Exploitation (INIMEX) project began in August 1997, and is associated with the DARPA MSTAR program. INIMEX focuses on geospatial workspaces for an image analyst, and initially used Pad++ but switched to Jazz in mid-1999.
  • The DARPA Dynamic Database (DDB) program uses INIMEX to view the contents of the DDB database. DDB expands the application domain from imagery to a variety of time-varying geo-spatial data types.
  • The DARPA Command Post of the Future (CPOF) program is exploring how to create a command-post decision making environment using Jazz.
  • The ARDA GI2VIS program is examining how to create an information analysis workspace using Jazz.

The following paper describes the original INIMEX work that focussed on geospatial ZUI surfaces using Pad++. Similar functionality now exists using Jazz.

  • R.D. Rimey, W.N. Klarquist, and C.H. Debrunner, "Design of Zoomable User Interfaces and Lens Tools for Intelligence Analysis," 27th AIPR Workshop: Advances in Computer-Assisted Recognition, R.J. Mericsko, ed., Proceedings of SPIE, Volume 3584, 1998, pages 17-28. [Download PDF file]
