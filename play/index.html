.Net Applets

The .NET applets require the .NET runtime version 1.1.  Also, the .NET applets only work with Internet Explorer 5.5 or later.

  • Fisheye Calendar - A simple tabular fisheye application.
  • Graph Editor - A simple graph drawing program.
  • Presentation Tool - An animated slide viewing program.
    Warning: This applet loads several images and may take a while to download, depending on your connection speed.
