Applications Built on Piccolo

Piccolo has been designed from the bottom up for use in robust, commercial software development.  This page shows several real-world applications that have been built on Piccolo.

As you browse through this section and see ongoing work or potential ideas for applications, please let us know if you're interested in pursuing some of your own. We can connect you with additional resources, talk about interface concepts, brainstorm about potential implementations.

We'd like to post examples of this work for the broader Piccolo community. If you have something that you can share, please contact us, and we'd be happy to add it to this pages.
